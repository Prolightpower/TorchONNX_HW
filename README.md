# 🍷 Vino Intelligence — Deep Learning Wine Quality Predictor

**PyTorch → ONNX → GitHub Pages**  
A regression Deep Neural Network that predicts red wine quality from physicochemical measurements, deployed entirely in the browser using ONNX Runtime Web.

---

## 📁 Repository Structure

```
├── index.html              ← GitHub Pages frontend (ONNX Runtime Web)
├── wine_quality_dnn.onnx   ← Trained ONNX model (generated by training script)
├── scaler_params.json      ← StandardScaler mean/std for frontend normalization
├── training_history.json   ← Loss/R² curves for visualization
├── train_and_export.py     ← PyTorch training + ONNX export script
└── README.md
```

---

## 🧠 Model Architecture

```
Input (11) → Linear(128) + BN + ReLU + Dropout(0.3)
           → Linear(256) + BN + ReLU + Dropout(0.3)
           → Linear(256) + BN + ReLU + Dropout(0.2)
           → Linear(128) + BN + ReLU + Dropout(0.2)
           → Linear(64)  + ReLU
           → Linear(1)   [regression output]
```

| Metric        | Value    |
|---------------|----------|
| Total Params  | 128,641  |
| Val MSE       | 0.459    |
| Val R²        | 0.740    |
| Optimizer     | AdamW    |
| Scheduler     | CosineAnnealing |
| Epochs        | 150      |

## 📊 Dataset

**UCI Red Wine Quality Dataset**  
- Source: https://archive.ics.uci.edu/dataset/186/wine+quality  
- 1,599 samples of Portuguese Vinho Verde red wine  
- 11 physicochemical input features  
- Target: quality score (integer 3–8)  
- **Different from** Iris, MNIST, Boston Housing, and other standard lab datasets

Features: fixed acidity, volatile acidity, citric acid, residual sugar, chlorides,
free sulfur dioxide, total sulfur dioxide, density, pH, sulphates, alcohol

## 🚀 Step-by-Step Deployment

### Step 1 — Train the model locally

```bash
# Install dependencies
pip install torch onnx onnxruntime numpy pandas scikit-learn

# Run training + ONNX export
python train_and_export.py
```

This generates:
- `wine_quality_dnn.onnx` — the model
- `scaler_params.json` — already in repo
- `training_history.json` — already in repo

### Step 2 — Push to GitHub

```bash
git init
git add index.html wine_quality_dnn.onnx scaler_params.json training_history.json train_and_export.py README.md
git commit -m "Deploy Wine Quality DNN"
git remote add origin https://github.com/YOUR_USERNAME/wine-quality-dnn.git
git push -u origin main
```

### Step 3 — Enable GitHub Pages

1. Go to **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** / root
4. Click **Save**

Your app will be live at:  
`https://YOUR_USERNAME.github.io/wine-quality-dnn/`

## 🔬 How Inference Works (Browser-Side)

```
User Input
    ↓
StandardScaler normalization (z = (x - mean) / std)
    ↓
Float32Array → ort.Tensor([1, 11])
    ↓
ort.InferenceSession.run({ features: tensor })  ← ONNX Runtime Web
    ↓
quality_score output
    ↓
Clamp to [3, 10] + display
```

No server required — 100% static, runs in the browser via WebAssembly.
