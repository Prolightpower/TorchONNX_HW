How It Works

1. Data loaded from UCI, scaled with StandardScaler (zero mean, unit variance).
2. 80/20 train/test split with random_state=42.
3. PyTorch trained 150 epochs with AdamW + CosineAnnealingLR.
4. Exported to using torch.onnx.export() at opset 11.
5. Browser loads the .onnx file with ONNX Runtime Web (WebAssembly).
6. User inputs are standardized with saved scaler params, fed to model, output clamped to [3, 10].
